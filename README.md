# 📊 Student Marks Analysis

## 📌 Project Overview
This project analyzes student marks using Microsoft Excel.  
It helps in understanding performance using basic statistical calculations.

---

## 🎯 Objective
- To calculate total marks of students  
- To find average performance  
- To identify highest and lowest scores  

---

## 🛠️ Tools Used
- Microsoft Excel  

---

## 📈 Analysis Performed
- Total Marks Calculation  
- Average Marks  
- Maximum Marks  
- Minimum Marks  

---

## 📂 Dataset
The dataset contains:
- Student Name  
- Subject  
- Marks  

---

## 📊 Key Insights
- Average marks of students is calculated  
- Highest scoring student identified  
- Lowest performance also analyzed  

---

## 🚀 How to Use
1. Open the Excel file `marks.xlsx`  
2. Check formulas used in the sheet  
3. View the chart for visualization  

---

## 📎 Project Files
- marks.xlsx → Contains dataset and analysis  

---

## 📊 Sample Output

Total Marks: 417  
Average Marks: 83.4  
Highest Marks: 90  
Lowest Marks: 76
